# Product definition

## Vision

Chronicle Forge turns permitted tabletop source material and campaign notes into
searchable, source-backed, permission-aware knowledge. It should provide genuine
private utility and demonstrate production-oriented full-stack and AI engineering.

## MVP boundary

- one demo campaign with GM and player roles
- safe document import with page-aware extraction
- rulebook and campaign-note document types
- authorized full-text search with citations
- source-backed AI answers
- hard exclusion of GM-only content from player retrieval
- reproducible local startup

Character management, VTT/MCP integrations, complex editors, multi-agent
orchestration, and self-hosted models remain outside the first release.

## Roadmap

1. `v0.1`: runnable skeleton and health checks
2. `v0.2`: document import and search projection
3. `v0.3`: permission-aware search
4. `v0.4`: cited AI answers and evaluation
